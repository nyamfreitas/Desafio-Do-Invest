# Desafio-Dio-Invest
Simulador de investimentos 
Esse desafio pode proporcionar 
uma compreensao maior das ferrametas do excel.
Dessa forma foi possivel gerar uma planilha funcional,
a qual simula investimentos.
Foi utilizado uma planilha do excel.
Apesar da complexidade, a repetição traz bastante conhecimento 
e confiança, com o tempo e uso nos sentimos familiarizados 
com as fórmulas e atalhos. 
O arquivo conta com duas planilhas. planilha 1 que conta
com várias tabelas calculadas e planilha 2 com dados de
apoio.
